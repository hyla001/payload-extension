## Quick Install

### Chrome
1. Buka `chrome://extensions`
2. Enable **Developer Mode** (toggle kanan atas)
3. Klik **Load unpacked**
4. Pilih folder `payload-extension`
5. Extension siap digunakan!

### Firefox
1. Buka `about:debugging#/runtime/this-firefox`
2. Klik **Load Temporary Add-on**
3. Pilih file `payload-extension/manifest.json`
4. Extension siap digunakan!
